CSS Syntax

A sintaxe consiste em um seletor e um bloco de declaração contento suas propriedades e valores. Exemplo:

h1 {
background-color: #000;
color: white;
}

No qual:

  • O h1 é o seletor;
  • Entre os semicolchetes é o bloco de declaração;
  • O background-color é uma propriedade e seu valor é #000;
  • E o color é uma propriedade e seu valor é white.
