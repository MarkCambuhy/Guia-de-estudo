Selectors

Como o próprio nome diz, ele seleciona qual tag em HTML você deseja estilizar. Exemplos de alguns seletores básicos:

/* Por elemento, ele seleciona todos os elementos <p>*/
p {
background-color: #000;
color: white;
}
/* Por class, ele seleciona todos os elementos com a class = "text"*/
.text {
background-color: #000;
color: white;
}
/* Por id, ele seleciona todos os elementos com o id = "mainText"*/
#maintext {
background-color: #000;
color: white;
}
/* Elemento universal, todos os elementos são selecionados*/
* {
margin: 0;
padding: 0;
box-sizing: border-box;
}
/* Seleciona apenas os elementos <p> com a class="maintext"*/
p.maintext {
background-color: blue;
color: green;
}
/* Seleciona os elementos, <p>, <div> e <span> ao mesmo tempo.*/
p, div, span{
margin: 0;
padding: 0;
box-sizing: border-box;
}
/* Todos os filhos de header que forem a serão selecionados.*/ header a {
color: red;
}
